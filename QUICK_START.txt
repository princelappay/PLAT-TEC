HOW TO RUN THE WEBSITE:
-------------------------------------------------

1. Open your web browser

2. Go to: http://localhost/platform-tech/

3. You'll see the login page

4. Click "Register" to create a new account
   - Choose any username and password
   - Submit the form

5. After registration, you'll be redirected to login
   - Enter your credentials
   - Login to access the dashboard

6. Explore the features:
   - Browse Cars: See all available vehicles
   - Filter by type (Sedan, SUV, Sports, etc.)
   - Sort by price or year
   - Book a car by selecting dates
   - View your bookings
   - Cancel bookings if needed

TROUBLESHOOTING:
-------------------------------------------------

If you see "Connection failed":
- Check if MySQL is running in XAMPP
- Verify database name is "vanguard_db"

If page doesn't load:
- Check if Apache is running in XAMPP
- Use correct URL: http://localhost/platform-tech/

If you can't login after registration:
- Make sure you registered successfully
- Check username and password are correct
